## 🚀 Featured Project

### 📱 [Multi-Provider Installment Calculator](https://github.com/DMStyles/installment-calculator) `v1.6.1`
A sleek, dark-mode financial utility built to calculate and compare Buy Now Pay Later (BNPL) plans across regional merchants (Koko, PayZy, MintPay) in Sri Lanka.

* **Beautiful UI/UX:** Eye-friendly premium dark mode with custom brand accent colors, glassmorphic card layouts, and press-scale tap micro-animations.
* **Server-Driven Push Notifications:** Migration from local exact alarms to a cloud-based cron scheduler running daily on Firebase Cloud Functions, notifying users of upcoming due dates in their local timezone (`Asia/Colombo`).
* **Cloud Syncing:** Built-in Firebase Firestore engine syncing device logs and custom notification lead-day preferences seamlessly.
* **Optimized footprint:** Size-optimized split APK release builds decreasing app download size from 54MB to 19.5MB for `arm64-v8a` architectures.
* **🛠️ Tech Stack:** Flutter, Dart, Firebase (FCM, Firestore, Cloud Functions), Node.js, Kotlin, GitHub Actions.
